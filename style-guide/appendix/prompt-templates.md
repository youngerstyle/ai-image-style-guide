# Prompt 模板集 Prompt Templates

本附录提供了各种风格的 Prompt 模板，可以直接复制使用或作为参考进行修改。

This appendix provides prompt templates for various styles that can be copied directly or modified as reference.

---

## 通用模板结构 Universal Template Structure

### 基础结构 Basic Structure

```
[主体描述], [风格类型], [媒介], [光影], [色彩], [氛围], [其他修饰]
[Subject], [style], [medium], [lighting], [colors], [mood], [other modifiers]
```

### 完整结构 Complete Structure

```
[主体描述], [动作/姿态], [风格类型], [艺术家参考], [媒介/技法],
[构图], [光影], [色彩], [氛围], [细节程度], [画质修饰]
```

## 修饰词速查 Modifier Quick Reference

### 画质 Quality
- `high quality`, `high detail`, `8K`, `4K`
- `masterpiece`, `best quality`, `professional`
- `sharp focus`, `crisp`, `clear`

### 光影 Lighting
- `dramatic lighting`, `soft lighting`, `natural light`
- `golden hour`, `blue hour`, `studio lighting`
- `rim light`, `backlit`, `volumetric light`

### 氛围 Mood
- `peaceful`, `dramatic`, `mysterious`, `romantic`
- `energetic`, `melancholic`, `whimsical`, `epic`
- `cozy`, `ethereal`, `dark`, `bright`

### 细节 Detail
- `highly detailed`, `intricate`, `elaborate`
- `simple`, `minimal`, `clean`
- `textured`, `smooth`, `rough`

### 构图 Composition
- `centered`, `rule of thirds`, `symmetrical`
- `dynamic angle`, `close-up`, `wide shot`
- `bird's eye view`, `worm's eye view`

## 负面提示词 Negative Prompts

常用的负面提示词，用于排除不想要的元素：

Common negative prompts to exclude unwanted elements:

```
低质量排除：
blurry, low quality, low resolution, pixelated,
jpeg artifacts, noise, grainy

变形排除：
deformed, distorted, disfigured, bad anatomy,
wrong proportions, extra limbs, missing limbs

风格排除（根据需要）：
photorealistic (当需要插画时)
cartoon (当需要写实时)
3D render (当需要2D时)
```
