# 📊 信息可视化完整词典

> **这是最重要的一章**：当你想用图像展示信息、流程、数据、知识时，来这里找关键词。

知道 `infographic`（信息图）这个词，就能打开一扇门。但这扇门后面还有几十个相关概念，每一个都能帮你生成不同类型的信息展示图。

---

## 🎯 核心痛点

你是不是经常遇到这些情况：

- 想做一个流程说明图，但不知道该叫什么
- 想展示数据对比，但生成的图不是你想要的
- 看到别人的知识卡片很好看，但不知道用什么关键词

**这一章就是解决这个问题的**。

---

## 📚 完整分类

### 一、流程/步骤类

当你想展示**从A到B的过程**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 信息图 | `infographic` | 图文结合的信息展示 | 知识科普、数据展示 |
| 路线图 | `roadmap` | 起点→终点的路径，有里程碑 | 项目规划、学习路径 |
| 旅程图 | `journey map`, `user journey` | 用户体验的完整流程 | 用户体验、服务设计 |
| 流程图 | `flowchart`, `process flow` | 步骤+箭头+判断分支 | 工作流程、决策流程 |
| 时间线 | `timeline` | 按时间顺序排列事件 | 历史、项目进度、个人经历 |
| 甘特图 | `gantt chart` | 横向时间条，显示任务进度 | 项目管理 |
| 泳道图 | `swimlane diagram` | 按角色/部门分列的流程 | 跨部门协作流程 |
| 循环图 | `cycle diagram` | 循环往复的流程 | 生命周期、迭代流程 |
| 漏斗图 | `funnel diagram` | 逐级递减的漏斗 | 销售转化、用户流失 |

**Prompt 示例**：
```
Create a hand-drawn roadmap infographic showing 5 steps to learn programming,
with a winding path from "beginner" to "expert",
numbered milestones, and a flag at the mountain peak destination.
Kraft paper texture background, sketchnote style.
```

### 二、结构/关系类

当你想展示**事物之间的关系**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 思维导图 | `mind map` | 中心主题向外发散 | 头脑风暴、知识整理 |
| 概念图 | `concept map` | 概念节点+关系连线 | 知识体系、概念关系 |
| 组织架构图 | `org chart`, `organization chart` | 层级树状结构 | 公司架构、团队结构 |
| 韦恩图 | `venn diagram` | 圆圈交集显示共同点 | 对比分析、集合关系 |
| 金字塔图 | `pyramid diagram` | 层级金字塔结构 | 需求层次、优先级 |
| 矩阵图 | `matrix diagram`, `2x2 matrix` | 四象限或多格矩阵 | 分类分析、决策矩阵 |
| 树状图 | `tree diagram` | 分支结构 | 分类、决策树 |
| 网络图 | `network diagram` | 节点+连线的网络 | 社交关系、系统架构 |
| 层级图 | `hierarchy diagram` | 上下层级关系 | 分类体系、权限结构 |
| 同心圆图 | `concentric circles` | 由内向外的圆环 | 核心-外围关系 |

**Prompt 示例**：
```
Create a colorful mind map with "AI Learning" at the center,
branching out to 5 main topics: Machine Learning, Deep Learning, NLP, Computer Vision, Reinforcement Learning.
Each branch has 3-4 sub-topics with small icons.
Clean white background, modern flat design style.
```

### 三、数据展示类

当你想展示**数字和统计**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 数据可视化 | `data visualization` | 数据的图形化展示 | 数据报告、分析结果 |
| 仪表盘 | `dashboard` | 多个指标的集合展示 | KPI监控、数据概览 |
| 饼图 | `pie chart` | 圆形占比展示 | 比例分布 |
| 柱状图 | `bar chart` | 柱形高度对比 | 数量对比 |
| 折线图 | `line chart` | 趋势变化曲线 | 时间趋势 |
| 雷达图 | `radar chart`, `spider chart` | 多维度对比 | 能力评估、产品对比 |
| 桑基图 | `sankey diagram` | 流量/转化的宽度变化 | 流量分析、能源流向 |
| 词云 | `word cloud` | 关键词大小表示频率 | 热词分析、主题提取 |
| 热力图 | `heatmap` | 颜色深浅表示数值 | 密度分布、相关性 |
| 气泡图 | `bubble chart` | 气泡大小表示数值 | 三维数据对比 |
| 面积图 | `area chart` | 填充的折线图 | 累计趋势 |
| 散点图 | `scatter plot` | 点的分布 | 相关性分析 |

**Prompt 示例**：
```
Create a modern dashboard infographic showing website analytics:
- Large number "1.2M" for monthly visitors
- Pie chart for traffic sources
- Line chart for growth trend
- Bar chart for top Dark theme with neon accent colors, clean minimal layout.
```

### 四、手绘/笔记类

当你想要**手绘风格的信息展示**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 涂鸦笔记 | `sketchnote` | 手绘图+文字的笔记 | 会议记录、读书笔记 |
| 视觉笔记 | `visual notes` | 图像化的笔记 | 学习笔记、思考整理 |
| 涂鸦风格 | `doodle style` | 随意的手绘线条 | 轻松、有趣的内容 |
| 白板风格 | `whiteboard style` | 像在白板上画的 | 讲解、教学 |
| 黑板风格 | `chalkboard style` | 粉笔画效果 | 教育、复古感 |
| 便签风格 | `sticky notes style` | 便利贴元素 | 头脑风暴、待办事项 |
| 手账风格 | `bullet journal style` | 手账本的排版风格 | 日程、计划、记录 |
| 线稿风格 | `line drawing style` | 纯线条无填充 | 简洁、技术感 |
| 草图风格 | `sketch style` | 草稿般的手绘 | 概念展示、初步想法 |

**Prompt 示例**：
```
Create a sketchnote style visual summary of "How to be productive":
- Hand-drawn icons and stick figures
- Handwritten-style text
- Arrows connecting ideas
- Sticky notes and speech bubbles
- Kraft paper or notebook paper background
Warm, friendly, educational feel.
```

### 五、教育/说明类

当你想**解释或教学**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 教学图解 | `instructional diagram` | 教学用的图解 | 教程、说明书 |
| 操作指南 | `how-to guide`, `tutorial` | 步骤说明 | 使用教程、DIY指南 |
| 对比图 | `comparison chart`, `vs diagram` | A vs B 的对比 | 产品对比、方案对比 |
| 清单图 | `checklist infographic` | 打勾清单形式 | 待办事项、检查清单 |
| 解剖图 | `anatomy diagram` | 分解展示内部结构 | 产品结构、生物解剖 |
| 剖面图 | `cross-section`, `cutaway` | 切开看内部 | 建筑、机械、地质 |
| 爆炸图 | `exploded view` | 零件分解展示 | 产品组装、机械结构 |
| 标注图 | `labeled diagram` | 带标签的说明图 | 部件说明、功能介绍 |
| 前后对比 | `before and after` | 变化前后对比 | 效果展示、改进说明 |
| 图解说明 | `illustrated explanation` | 图文配合的解释 | 概念解释、原理说明 |

**Prompt 示例**：
```
Create an exploded view diagram of a mechanical watch,
showing all internal components separated and labeled:
case, dial, hands, movement, gears, spring, crown.
Technical illustration style, clean white background,
thin lines connecting parts to labels.
```

### 六、商业/营销类

当你想做**商业用途的信息展示**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 一页纸 | `one-pager` | 单页信息汇总 | 产品介绍、项目概述 |
| 宣传单页 | `flyer`, `leaflet` | 营销传单 | 活动宣传、产品推广 |
| 速查表 | `cheat sheet` | 快速参考卡片 | 快捷键、公式、语法 |
| 知识卡片 | `knowledge card` | 单个知识点展示 | 学习卡片、科普 |
| 事实清单 | `fact sheet` | 关键事实列表 | 产品参数、公司介绍 |
| 统计图 | `statistics infographic` | 数字+图形结合 | 数据报告、年度总结 |
| 简历图 | `resume infographic` | 可视化简历 | 求职、个人介| 报价单 | `pricing table` | 价格对比表格 | 产品定价、套餐对比 |
| 功能对比 | `feature comparison` | 功能点对比 | 产品对比、方案选择 |
| 里程碑 | `milestone infographic` | 重要节点展示 | 公司历史、项目进展 |

**Prompt 示例**：
```
Create a modern one-pager for a SaaS product:
- Hero section with product name and tagline
- 3 key features with icons
- Pricing table with 3 tiers
- Customer testimonial quote
- Call-to-action button
Clean, professional, blue and white color scheme.
```

### 七、地图/空间类

当你想展示**空间或位置关系**时：

| 概念 | 英文关键词 | 特点 | 适用场景 |
|-----|-----------|------|---------|
| 示意地图 | `illustrated map` | 手绘风格地图 | 旅游指南、活动地图 |
| 楼层平面图 | `floor plan` | 空间布局俯视图 | 室内设计、场馆导览 |
| 生态系统图 | `ecosystem map` | 关系网络图 | 商业生态、技术栈 |
| 用户故事地图 | `user story map` | 产品功能规划图 | 产品设计、敏捷开发 |
| 客户旅程地图 | `customer journey map` | 客户体验全流程 | 服务设计、体验优化 |
| 服务蓝图 | `service blueprint` | 服务流程全景图 | 服务设计 |
| 信息架构图 | `information architecture` | 网站/App结构图 | 产品设计、网站规划 |

## 🎨 风格修饰词

这些词可以和上面任何概念**组合使用**，改变视觉风格：

### 绘制风格

| 风格 | 英文 | 效果 |
|-----|------|------|
| 手绘风 | `hand-drawn`, `hand-sketched` | 手绘线条感 |
| 扁平风 | `flat design` | 无阴影、纯色块 |
| 等距风 | `isometric` | 2.5D 立体视角 |
| 极简风 | `minimalist` | 简洁、留白多 |

| 图标风 | `icon-based`, `iconographic` | 以图标为主 |
| 卡通风 | `cartoon style` | 卡通化 |
| 3D风 | `3D style`, `3D render` | 立体渲染 |

### 材质/背景

| 风格 | 英文 | 效果 |

|  | `kraft paper texture` | 牛皮纸质感 |

## 💡 组合公式

**基础公式**：
```
[风格修饰词] + [信息类型] + [内容描述] + [背景/材质]
```

**示例组合**：

```
hand-drawn sketchnote style timeline showing the history of AI
on kraft paper texture background
```
→ 手绘涂鸦风格的AI历史时间线，牛皮纸背景

```
minimalist flat design comparison chart of iPhone vs Android
clean white background with blue accent colors
```
→ 极简扁平风格的iPhone vs Android对比图

```
isometric 3D style customer journey map for e-commerce shopping
colorful with soft shadows
```
→ 等距3D风格的电商购物旅程图

```
vintage chalkboard style mind map about coffee brewing methods
white chalk on dark green background
```
→ 复古黑板风格的咖啡冲泡方法思维导图

```
modern dashboard infographic showing social media analytics
dark theme with neon accent colors
```
→ 现代仪表盘风格的社交媒体数据分析图
