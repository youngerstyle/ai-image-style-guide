## 🚀 快速上手

### 场景1：我想把照片变成 Q 版形象

**关键词**：`chibi`, `3D chibi`, `Q版`, `cute chibi-style`

**万能模板**：
```
Transform the person in the photo into a cute chibi-style 3D character.
[场景描述]
The character should be in chibi 3D style, while the environment should be realistic.
```

Create a hand-drawn style infographic card in vertical 9:16 ratio.

## 💡 使用技巧

### 1. 先确定「图片类型」

这是最重要的一步！图片类型错了，细节再好也白搭。

❌ 错误示范：
```
一个可爱的女孩，大眼睛，粉色头发...（描述了一堆细节，但没说是什么类型的图）
```

✅ 正确示范：
```
A chibi-style 3D character, cute girl with big eyes and pink hair...
（先说是 chibi-style 3D，再描述细节）
```

### 2. 关键词放前面

AI 对 prompt 开头的词更敏感，把最重要的风格词放最前面。

```
chibi-style 3D character, [其他描述]...
infographic card, [其他描述]...
pixel art style, [其他描述]...
```

### 3. 善用案例的 prompt

本手册的每个风格都附带了真实可用的 prompt，直接复制修改比从零开始写效率高 10 倍。
