## 风格条目格式 Style Entry Format

每个风格条目包含以下信息：

Each style entry contains the following information:

```
## 风格名称 English Name

**别名**：其他常用名称

**简介**：风格的定义和核心特征

**视觉特征**：
- 特征 1
- 特征 2
- 特征 3

**历史背景**：风格的起源和发展

**代表艺术家/作品**：著名的代表

**变体与相关风格**：相似或衍生的风格

**Prompt 关键词**：可直接使用的英文关键词

**示例 Prompt**：完整的示例提示词

**适用场景**：这个风格适合用在什么地方
```

## 使用技巧 Usage Tips

### 快速查找 Quick Search

1. 使用浏览器的 `Ctrl+F` (Windows) 或 `Cmd+F` (Mac) 搜索关键词
2. 在字母索引中按首字母查找
3. 根据你的需求选择"按目的浏览"

### 组合使用 Combining Styles

- 可以从不同分类中选择风格进行组合
- 参考每个条目的"变体与相关风格"找到可搭配的风格
- 使用"Prompt 关键词"直接复制到你的提示词中

### 学习路径 Learning Path

如果你是新手，建议：
1. 先完成[教程轨](../tutorial/00-前言.md)的学习
2. 然后按兴趣浏览词典轨
3. 在实践中不断查阅和学习
