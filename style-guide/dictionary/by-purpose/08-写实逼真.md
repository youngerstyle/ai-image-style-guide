# 写实逼真 Realistic and Photorealistic Styles

当你想要创造真实、逼真、照片级的图像时，可以参考以下风格。

When you want to create realistic, lifelike, photorealistic images, refer to these styles.

---

## 绘画写实 Painting Realism

### 超写实主义 Hyperrealism
**关键词**：`hyperrealism`, `hyperrealistic`, `photorealistic painting`, `ultra-detailed`

**示例 Prompt**：
```
hyperrealistic oil painting, portrait,
every pore and hair visible, lifelike eyes,
indistinguishable from photograph,
museum quality, incredible detail
```

### 照片写实主义 Photorealism
**关键词**：`photorealism`, `photorealist`, `photo-realistic painting`, `Chuck Close style`

**示例 Prompt**：
```
photorealist painting, urban street scene,
painted to look exactly like a photograph,
reflections in windows, precise detail,
American photorealism style
```

### 古典写实 Classical Realism
**关键词**：`classical realism`, `academic realism`, `traditional realism`, `old master technique`

**示例 Prompt**：
```
classical realist portrait, oil painting,
old master technique, rich colors,
accurate anatomy, dramatic lighting,
Rembrandt influenced, timeless
```

### 当代写实 Contemporary Realism
**关键词**：`contemporary realism`, `modern realism`, `realistic painting`, `figurative`

**示例 Prompt**：
```
contemporary realist painting, figure study,
natural lighting, accurate proportions,
modern subject matter, painterly yet realistic
```

## 摄影写实 Photography Realism

### 专业人像 Professional Portrait
**关键词**：`professional portrait`, `studio photography`, `headshot`, `commercial portrait`

**示例 Prompt**：
```
professional studio portrait, perfect lighting,
sharp focus on eyes, clean background,
commercial quality, magazine cover worthy,
flawless skin detail
```

### 商业摄影 Commercial Photography
**关键词**：`commercial photography`, `advertising`, `product shot`, `professional`

**示例 Prompt**：
```
commercial product photography, luxury watch,
perfect studio lighting, reflective surface,
advertising quality, every detail sharp,
high-end commercial shoot
```

### 纪实摄影 Documentary Photography
**关键词**：`documentary`, `photojournalism`, `candid`, `authentic`, `real life`

**示例 Prompt**：
```
documentary photography, authentic moment,
natural light, unposed, real emotion,
photojournalistic style, storytelling
```

### 微距摄影 Macro Photography
**关键词**：`macro`, `extreme close-up`, `microscopic detail`, `magnified`

**示例 Prompt**：
```
macro photography, insect eye extreme close-up,
incredible detail, shallow depth of field,
scientific accuracy, nature's complexity revealed
```

## 写实 vs 风格化 Realism vs Stylization

| 写实 Realistic | 风格化 Stylized |
|---------------|-----------------|
| 追求真实 | 追求表达 |
| 准确的比例 | 夸张的比例 |
| 自然的色彩 | 艺术化的色彩 |
| 精确的细节 | 简化或强调的细节 |
| 物理正确 | 艺术自由 |

**何时选择写实**：
- 产品展示
- 建筑可视化
- 人像摄影
- 科学插图
- 商业广告
