# 暗黑神秘 Dark and Mysterious Styles

当你想要创造神秘、暗黑、哥特风格的图像时，可以参考以下风格。

When you want to create mysterious, dark, gothic images, refer to these styles.

---

## 艺术风格 Art Styles

### 哥特风格 Gothic
**关键词**：`gothic`, `Gothic art`, `dark gothic`, `medieval gothic`, `Gothic architecture`

**示例 Prompt**：
```
gothic cathedral interior, dramatic shadows,
stained glass windows, candlelight,
dark and mysterious atmosphere,
medieval gothic architecture
```

### 暗黑艺术 Dark Art
**关键词**：`dark art`, `dark fantasy`, `macabre`, `sinister`, `ominous`

**示例 Prompt**：
```
dark fantasy illustration, ominous castle,
stormy sky, lightning, ravens,
macabre atmosphere, sinister and foreboding
```

### 超现实主义暗黑 Dark Surrealism
**关键词**：`dark surrealism`, `nightmare`, `Beksiński style`, `disturbing`, `otherworldly`

**示例 Prompt**：
```
dark surrealist painting, Beksiński inspired,
nightmarish landscape, skeletal structures,
eerie atmosphere, disturbing beauty,
otherworldly and haunting
```

### 表现主义 Expressionism
**关键词**：`expressionism`, `dark expressionist`, `distorted`, `emotional`, `Munch style`

**示例 Prompt**：
```
expressionist painting, distorted figure,
dark emotional atmosphere, bold colors,
psychological tension, Edvard Munch inspired
```

## 摄影风格 Photography Styles

### 低调摄影 Low Key Photography
**关键词**：`low key`, `dark`, `dramatic shadows`, `chiaroscuro`, `moody`

**示例 Prompt**：
```
low key portrait, face emerging from darkness,
single light source, deep dramatic shadows,
mysterious and moody, film noir style
```

### 黑色电影 Film Noir
**关键词**：`film noir`, `noir`, `black and white noir`, `detective`, `shadows`

**示例 Prompt**：
```
film noir scene, detective in shadows,
venetian blind lighting, cigarette smoke,
black and white, mysterious femme fatale,
classic Hollywood noir atmosphere
```

### 暗黑人像 Dark Portrait
**关键词**：`dark portrait`, `moody portrait`, `shadowy`, `mysterious subject`

**示例 Prompt**：
```
dark moody portrait, half face in shadow,
intense gaze, dramatic side lighting,
mysterious and captivating, artistic
```

## 插画风格 Illustration Styles

### 暗黑插画 Dark Illustration
**关键词**：`dark illustration`, `horror illustration`, `creepy`, `eerie`, `sinister`

**示例 Prompt**：
```
dark fantasy illustration, haunted forest,
twisted trees, glowing eyes in darkness,
eerie fog, creepy atmosphere,
detailed and atmospheric
```

### 恐怖艺术 Horror Art
**关键词**：`horror art`, `horror illustration`, `scary`, `terrifying`, `nightmare`

**示例 Prompt**：
```
horror art, creature emerging from shadows,
terrifying design, dark atmosphere,
Lovecraftian influence, nightmare fuel
```

### 哥特插画 Gothic Illustration
**关键词**：`gothic illustration`, `Victorian gothic`, `dark romantic`, `Edgar Allan Poe`

**示例 Prompt**：
```
gothic illustration, Victorian mansion,
full moon, ravens, bare trees,
Edgar Allan Poe inspired, dark romantic,
mysterious and atmospheric
```

## 特殊暗黑风格 Special Dark Styles

### 赛博朋克暗黑 Dark Cyberpunk
**关键词**：`dark cyberpunk`, `cyberpunk noir`, `dystopian`, `gritty future`

**示例 Prompt**：
```
dark cyberpunk scene, rain-soaked alley,
neon signs flickering, shadowy figure,
dystopian atmosphere, gritty and noir
```

### 暗黑奇幻 Dark Fantasy
**关键词**：`dark fantasy`, `grimdark`, `dark medieval`, `Souls-like`, `Berserk style`

**示例 Prompt**：
```
dark fantasy warrior, corrupted armor,
ominous battlefield, dark souls inspired,
grimdark atmosphere, epic and foreboding
```

### 神秘学/神秘主义 Occult/Mysticism
**关键词**：`occult`, `mystical`, `esoteric`, `arcane`, `ritual`, `symbols`

**示例 Prompt**：
```
occult ritual scene, arcane symbols,
candlelit chamber, mysterious robed figures,
esoteric atmosphere, dark mysticism
```

### 死亡美学 Death Aesthetic
**关键词**：`memento mori`, `vanitas`, `skull`, `mortality`, `dark beauty`

**示例 Prompt**：

```
candle burning low, hourglass,
memento mori theme, dark beauty,

```
