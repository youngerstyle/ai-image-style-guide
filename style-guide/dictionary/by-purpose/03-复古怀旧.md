# 复古怀旧 Vintage and Nostalgic Styles

当你想要创造复古、怀旧、有年代感的图像时，可以参考以下风格。

When you want to create vintage, nostalgic, retro images, refer to these styles.

---

## 年代风格 Era Styles

### 1920s 爵士时代
**关键词**：`1920s`, `Art Deco`, `Jazz Age`, `Gatsby`, `flapper`
**特点**：装饰艺术、几何图案、华丽优雅

### 1950s 中世纪现代
**关键词**：`1950s`, `mid-century`, `atomic age`, `retro diner`
**特点**：乐观明亮、原子时代、美国梦

### 1960s 迷幻时代
**关键词**：`1960s`, `psychedelic`, `mod`, `flower power`
**特点**：迷幻色彩、波普艺术、反文化

### 1970s 迪斯科时代
**关键词**：`1970s`, `disco`, `groovy`, `earth tones`
**特点**：大地色调、迪斯科、放克

### 1980s 霓虹时代
**关键词**：`1980s`, `synthwave`, `neon`, `outrun`, `VHS`
**特点**：霓虹色彩、合成器美学、科技乐观

### 1990s 垃圾摇滚
**关键词**：`1990s`, `grunge`, `lo-fi`, `90s aesthetic`
**特点**：低保真、垃圾摇滚、早期互联网

---

## 摄影复古 Photography Vintage

### 胶片摄影 Film Photography
**关键词**：`film photography`, `35mm`, `analog`, `film grain`, `Kodak`, `Fuji`

**示例 Prompt**：
```
vintage film photography, shot on Kodak Portra,
warm skin tones, natural film grain,
nostalgic atmosphere, analog aesthetic
```

### 宝丽来 Polaroid
**关键词**：`Polaroid`, `instant film`, `Instax`, `white border`, `faded`

**示例 Prompt**：
```
Polaroid instant photo, white border frame,
slightly faded colors, casual snapshot feel,
nostalgic summer memory
```

### 老照片 Old Photograph
**关键词**：`old photo`, `vintage photograph`, `sepia`, `aged`, `antique`

**示例 Prompt**：
```
antique photograph style, sepia tones,
aged paper texture, slight damage and wear,
Victorian era portrait, historical feel
```

### 黑白复古 Vintage Black and White
**关键词**：`vintage black and white`, `classic B&W`, `film noir`, `old Hollywood`

---

## 设计复古 Design Vintage

### 复古海报 Vintage Poster
**关键词**：`vintage poster`, `retro poster`, `travel poster`, `propaganda poster`

**示例 Prompt**：
```
vintage travel poster, Visit Japan,
Art Deco style, bold flat colors,
elegant typography, 1930s aesthetic,
romantic and adventurous
```

### 复古广告 Vintage Advertisement
**关键词**：`vintage ad`, `retro advertisement`, `1950s commercial`, `old magazine ad`

**示例 Prompt**：
```
1950s vintage advertisement, happy housewife,
bright optimistic colors, retro typography,
American dream aesthetic, nostalgic
```

### 复古包装 Vintage Packaging
**关键词**：`vintage packaging`, `retro label`, `old brand`, `classic design`

## 特殊复古风格 Special Vintage Styles

### 蒸汽波 Vaporwave
**关键词**：`vaporwave`, `aesthetic`, `pink purple gradient`, `Greek statue`, `Japanese text`

**示例 Prompt**：
```
vaporwave aesthetic, pink and purple gradient,
Greek marble bust, palm trees, retro sun,
Japanese katakana text, 80s nostalgia,
dreamy and surreal
```

### 合成波 Synthwave
**关键词**：`synthwave`, `retrowave`, `outrun`, `neon grid`, `80s sunset`

**示例 Prompt**：
```
synthwave landscape, neon sunset gradient,
chrome retro car, perspective grid,
palm tree silhouettes, 80s nostalgia,
outrun aesthetic
```

### 复古未来主义 Retro-Futurism
**关键词**：`retro-futurism`, `retro-futuristic`, `vintage sci-fi`, `atomic age future`

**示例 Prompt**：
```
retro-futuristic scene, 1950s vision of future,
flying cars, dome cities, ray guns,
optimistic space age, chrome and pastels
```

### 蒸汽朋克 Steampunk
**关键词**：`steampunk`, `Victorian`, `brass`, `gears`, `steam-powered`

**示例 Prompt**：
```
steampunk aesthetic, Victorian inventor,
brass gears and copper pipes, goggles,
warm sepia tones, industrial fantasy
```

- `old-fashioned`, `antique`, `heritage`
