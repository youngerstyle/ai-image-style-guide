# 插画风格 Illustration Styles

本章收录了主要的插画风格，从传统插画到数字插画，从商业插画到艺术插画。

This chapter covers major illustration styles, from traditional to digital, from commercial to artistic.

---

## 日系动漫 Anime/Manga

**别名**：动漫风格、日本动画风格

**简介**：
日系动漫是源自日本的独特插画和动画风格，以大眼睛、精细的头发、鲜艳的色彩和独特的表情系统著称。

**视觉特征**：
- 大而富有表现力的眼睛
- 精细的头发绘制，常有高光
- 简化但富有表现力的面部特征
- 鲜艳的色彩
- 赛璐璐风格的阴影（硬边阴影）
- 清晰的轮廓线

**子风格 Sub-styles**：

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 少年漫画 Shonen | 动态、热血、战斗 | `shonen anime`, `action anime`, `dynamic` |
| 少女漫画 Shoujo | 浪漫、华丽、大眼睛 | `shoujo manga`, `romantic`, `sparkly eyes` |
| 萌系 Moe | 可爱、大眼、萌化 | `moe`, `cute anime`, `kawaii anime` |
| 写实动漫 Realistic Anime | 更写实的比例和细节 | `realistic anime`, `semi-realistic` |

**Prompt 关键词**：
`anime style`, `manga style`, `Japanese animation`, `cel shading`, `anime character`, `detailed eyes`, `anime art`

**示例 Prompt**：
```
anime style girl, detailed sparkling eyes,
flowing colorful hair, school uniform,
cherry blossom background, cel shading,
high quality anime illustration, vibrant colors
```

## 美式卡通 American Cartoon

**简介**：
美式卡通风格以夸张的表情、流畅的动态和幽默感著称，从迪士尼到现代动画都有其独特的表现。

**视觉特征**：
- 夸张的表情和动作
- 简化但富有表现力的形状
- 流畅的线条
- 鲜明的色彩
- 幽默感和个性

**子风格 Sub-styles**：

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 迪士尼风格 Disney | 经典、优雅、音乐剧感 | `Disney style`, `Disney animation` |
| 皮克斯风格 Pixar | 3D卡通、表情丰富 | `Pixar style`, `3D cartoon` |
| 卡通网络 Cartoon Network | 现代、几何、个性 | `Cartoon Network style`, `modern cartoon` |
| 成人动画 Adult Animation | 讽刺、粗犷 | `adult animation`, `Rick and Morty style` |

**Prompt 关键词**：
`cartoon style`, `American cartoon`, `Disney style`, `Pixar style`, `animated`, `expressive`, `cartoon character`

**示例 Prompt**：
```
Pixar style 3D character, friendly robot,
big expressive eyes, rounded shapes,
warm lighting, subsurface scattering,
animated movie quality, appealing design
```

## 欧洲漫画 European Comics

**别名**：BD风格（Bande Dessinée）

**简介**：
欧洲漫画有其独特的艺术传统，从法比漫画到英国漫画，风格多样且艺术性强。

**视觉特征**：
- 精细的线条和阴影
- 写实或半写实的比例
- 丰富的背景细节
- 独特的色彩运用
- 艺术性强

**Prompt 关键词**：
`European comics`, `BD style`, `Franco-Belgian comics`, `Moebius style`, `ligne claire`, `Tintin style`

**示例 Prompt**：
```
European comic style illustration,
ligne claire technique, clean precise lines,
Tintin inspired, detailed background,
vintage color palette, adventure scene
```

## 儿童插画 Children's Illustration

**简介**：
专为儿童读物设计的插画风格，强调温暖、友好、易于理解的视觉表达。

**视觉特征**：
- 温暖友好的色彩
- 简化的形状
- 可爱的角色设计
- 清晰易读的构图
- 富有想象力的场景

**Prompt 关键词**：
`children's book illustration`, `kids illustration`, `storybook`, `whimsical`, `friendly`, `warm colors`, `picture book`

**示例 Prompt**：
```
children's book illustration, cute forest animals,
warm friendly colors, simple rounded shapes,
whimsical storybook style, cozy atmosphere,
bedtime story feeling, heartwarming scene
```

## 时尚插画 Fashion Illustration

**简介**：
用于时尚行业的插画风格，强调服装、姿态和优雅感。

**视觉特征**：
- 修长的人体比例（通常9-10头身）
- 优雅的姿态
- 强调服装细节
- 艺术化的表达
- 流畅的线条

**Prompt 关键词**：
`fashion illustration`, `fashion sketch`, `haute couture`, `elongated figure`, `elegant pose`, `runway style`

**示例 Prompt**：
```
fashion illustration, elegant model,
elongated proportions, flowing evening gown,
watercolor style, loose brushstrokes,
haute couture, Parisian fashion week
```

## 概念艺术 Concept Art

**简介**：
为电影、游戏、动画等项目创作的设计图，用于确定视觉风格和设计方向。

**视觉特征**：
- 氛围营造强
- 快速但有效的表达
- 注重设计和功能
- 电影感构图
- 专业品质

**子风格 Sub-styles**：

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 角色概念 Character Concept | 角色设计、多角度 | `character concept`, `character design sheet` |
| 环境概念 Environment Concept | 场景设计、氛围 | `environment concept`, `scene design` |
| 道具概念 Prop Concept | 物品设计、细节 | `prop design`, `weapon design` |
| 生物概念 Creature Concept | 生物设计 | `creature design`, `monster concept` |

**Prompt 关键词**：
`concept art`, `concept design`, `visual development`, `production art`, `cinematic`, `professional concept`

**示例 Prompt**：
```
concept art, epic fantasy castle,
dramatic lighting, atmospheric perspective,
cinematic composition, matte painting style,
professional quality, movie production design
```

## 科学插画 Scientific Illustration

**简介**：
用于科学、医学、自然历史等领域的精确插画，性。

**视觉特征**：
- 高度准确
- 清晰的细节
- 教育目的
- 通常为白色背景
- 专业的标注

**Prompt 关键词**：
`scientific illustration`, `botanical illustration`, `anatomical`, `naturalist`, `educational`, `technical accuracy`

**示例 Prompt**：
```
scientific botanical illustration,
detailed flower anatomy, accurate colors,
white background, naturalist style,
educational diagram, museum quality
```

## 社论插画 Editorial Illustration

**简介**：
用于杂志、报纸、文章配图的概念性插画，强调观点表达和视觉隐喻。

**视觉特征**：
- 概念性强
- 视觉隐喻
- 风格多样
- 传达观点
- 引人思考

**Prompt 关键词**：
`editorial illustration`, `conceptual`, `magazine illustration`, `metaphorical`, `thought-ping`

**示例 Prompt**：
```
editorial illustration about technology and privacy,
conceptual metaphor, person made of data,
bold graphic style, limited color palette,
thought-provoking imagery, magazine cover quality
```

## 线条艺术 Line Art

**简介**：
纯线条表现的插画风格，不使用或极少使用填充色。

**视觉特征**：
- 纯线条表现
- 无填充或极少填充
- 强调轮廓和细节
- 黑白或单色
- 精细或简约

**Prompt 关键词**：
`line art`, `linework`, `ink drawing`, `outline`, `black and white`, `pen and ink`, `contour drawing`

**示例 Prompt**：
```
detailed line art portrait, fine pen strokes,
intricate linework, no color fill,
black ink on white, elegant contours,
artistic and precise
```

---

## 扁平插画 Flat Illustration

**简介**：
使用扁平色块、无渐变的现代插画风格，常用于UI设计和商业插画。

**视觉特征**：
- 纯色块
- 无渐变或阴影
- 简化的形状
- 几何化
- 现代感

**Prompt 关键词**：
`flat illustration`, `flat design`, `vector style`, `geometric`, `simple shapes`, `no gradients`, `modern illustration`

**示例 Prompt**：
```
flat illustration, people working in office,
simple geometric shapes, solid colors,
no gradients or shadows, modern vector style,
clean and minimal, corporate illustration
```

## 复古插画 Vintage Illustration

**简介**：
模仿过去时代插画风格的作品，常见于海报、广告和书籍。

**视觉特征**：
- 复古色调
- 特定时代的风格特征
- 手绘质感
- 印刷纹理
- 怀旧感

**Prompt 关键词**：
`vintage illustration`, `retro illustration`, `1950s style`, `mid-century`, `nostalgic`, `classic illustration`

**示例 Prompt**：
```
vintage 1950s advertisement illustration,
happy family scene, retro color palette,
mid-century modern style, hand-painted look,
nostalgic American dream aesthetic
```
