# 3D与数字艺术 3D & Digital Art

本章收录各种3D渲染和数字艺术风格。

---

## 写实3D / Photorealistic 3D

### 关键词
`photorealistic 3D`, `realistic render`, `CGI`, `hyperrealistic 3D`

### 特点
- 接近真实照片的3D渲染效果
- 精细的材质、光影和细节
- 常用于产品展示、建筑可视化

### Prompt 示例
```
Photorealistic 3D render of [主体],
studio lighting, high detail, 8K resolution
```

---

## 低多边形 / Low Poly

### 关键词
`low poly`, `low polygon`, `geometric 3D`, `faceted`

### 特点
- 由少量多边形面组成
- 几何感强，有棱角
- 常用于游戏、插画、头像

### Prompt 示例
```
Low poly 3D style [主体],
geometric facets, soft gradient colors, clean background
```

---

## 体素艺术 / Voxel Art

### 关键词
`voxel`, `voxel art`, `3D pixel`, `Minecraft style`

### 特点
- 3D版的像素艺术
- 由立方体方块组成
- 类似《我的世界》风格

### Prompt 示例
```
Voxel art style [主体],
isometric view, colorful blocks, clean render
```

---

## 粘土风格 / Clay Render

### 关键词
`clay render`, `clay style`, `plasticine`, `claymation`

### 特点
- 像粘土/橡皮泥制作的效果
- 柔和的表面，无纹理
- 可爱、手工感

### Prompt 示例
```
Clay render style [主体],
soft matte surface, pastel colors, studio lighting
```

---

## 等距视角 / Isometric

### 关键词
`isometric`, `isometric view`, `2.5D`, `dimetric`

### 特点
- 45度俯视角度
- 无透视变形
- 常用于游戏、图标、信息图

### Prompt 示例
```
Isometric 3D illustration of [主体],
clean lines, vibrant colors, white background
```

---

## 卡通渲染 / Toon Shading

### 关键词
`toon shading`, `cel shading`, `cartoon 3D`, `anime 3D`

### 特点
- 3D模型+卡通着色
- 明显的色块分界
- 动漫/卡通质感

### Prompt 示例
```
Toon shaded 3D character,
cel shading style, bold outlines, anime aesthetic
```

---

## 霓虹风格 / Neon Style

### 关键词
`neon`, `neon glow`, `neon lights`, `glowing`

### 特点
- 发光的霓虹灯效果
- 通常配合深色背景
- 赛博朋克常用元素

### Prompt 示例
```
Neon glowing [主体],
dark background, vibrant neon colors, reflection
```

---

## 全息效果 / Holographic

### 关键词
`holographic`, `hologram`, `iridescent`, `rainbow reflection`

### 特点
- 彩虹色反射效果
- 半透明、科技感
- 未来主义风格

### Prompt 示例
```
Holographic 3D [主体],
iridescent colors, transparent effect, futuristic
```

---

## 玻璃质感 / Glass Material

### 关键词
`glass`, `crystal`, `transparent`, `translucent`, `frosted glass`

### 特点
- 透明或半透明材质
- 折射、反射效果
- 优雅、现代感

### Prompt 示例
```
3D glass sculpture of [主体],
transparent material, light refraction, clean background
```

---

## 金属质感 / Metallic

### 关键词
`metallic`, `chrome`, `gold`, `silver`, `brushed metal`

### 特点
- 金属反射效果
- 可以是镜面或拉丝
- 高级、工业感

### Prompt 示例
```
Chrome metallic 3D [主体],
mirror reflection, studio lighting, dark background
```

---

## 线框渲染 / Wireframe

### 关键词
`wireframe`, `wireframe render`, `mesh`, `3D grid`

### 特点
- 只显示3D模型的线框
- 技术感、科技感
- 常用于展示结构

### Prompt 示例
```
Wireframe 3D render of [主体],
blue lines on dark background, technical style
```

---

## 点云 / Point Cloud

### 关键词
`point cloud`, `particle`, `dot matrix 3D`

### 特点
- 由点组成的3D形态
- 科技感、数据感
- 常用于扫描可视化

### Prompt 示例
```
Point cloud visualization of [主体],
scattered particles, gradient colors, dark background
```
