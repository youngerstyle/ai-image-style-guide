# 媒介与技法 Medium and Techniques

## 什么是媒介 What is Medium

**媒介（Medium）** 是指创作艺术作品所使用的材料和工具。不同的媒介有其独特的视觉特征，这些特征在 AI 绘图中可以被模拟和再现。

**Medium** refers to the materials and tools used to create artwork. Different media have unique visual characteristics that can be simulated and reproduced in AI art.

---

## 传统绘画媒介 Traditional Painting Media

### 油画 Oil Painting

**视觉特征 Visual Characteristics**：
- 丰富的色彩层次
- 可见的笔触
- 厚重的质感（厚涂技法）
- 光泽感

**历史背景 Historical Context**：
文艺复兴以来西方绑画的主要媒介，达芬奇、伦勃朗、梵高等大师的首选。

**Prompt 关键词**：
`oil painting`, `oil on canvas`, `painterly`, `visible brushstrokes`, `impasto`, `glazing technique`

**示例 Example**：
```
oil painting portrait, rich colors, visible brushstrokes,
classical technique, Rembrandt lighting, museum quality
```

---

### 水彩 Watercolor

**视觉特征 Visual Characteristics**：
- 透明、轻盈
- 颜色渗透和晕染
- 留白技法
- 柔和的边缘

**历史背景 Historical Context**：
常用于风景画、植物插画，以其轻盈透明的特质著称。

**Prompt 关键词**：
`watercolor`, `watercolour`, `aquarelle`, `wet-on-wet`, `transparent washes`, `soft edges`, `paper texture`

**示例 Example**：
```
watercolor landscape, soft washes of color,
wet-on-wet technique, visible paper texture,
delicate and ethereal
```

---

### 水粉/丙烯 Gouache/Acrylic

**视觉特征 Visual Characteristics**：
- 不透明
- 平涂效果好
- 色彩鲜艳
- 快干

**Prompt 关键词**：
`gouache`, `acrylic painting`, `opaque colors`, `flat colors`, `matte finish`

**示例 Example**：
```
gouache illustration, flat opaque colors,
bold color blocks, matte finish, vintage poster style
```

---

### 素描 Drawing/Sketch

**视觉特征 Visual Characteristics**：
- 线条为主
- 黑白或有限色彩
- 交叉阴影
- 纸张质感

| 类型 Type | 特征 Features | 关键词 Keywords |
|----------|---------------|-----------------|
| 铅笔素描 Pencil | 柔和、可擦除、灰度 | `pencil drawing`, `graphite`, `pencil sketch` |
| 炭笔素描 Charcoal | 深黑、粗犷、有质感 | `charcoal drawing`, `charcoal sketch`, `dramatic` |
| 钢笔素描 Pen/Ink | 清晰、永久、高对比 | `pen and ink`, `ink drawing`, `pen sketch` |
| 彩色铅笔 Colored Pencil | 柔和色彩、纹理 | `colored pencil`, `pencil crayon` |

**示例 Example**：
```
detailed pencil portrait, graphite on paper,
realistic shading, fine cross-hatching, photorealistic
```

---

### 版画 Printmaking

**视觉特征 Visual Characteristics**：
- 独特的印刷质感
- 有限的色彩层次
- 复制性

| 类型 Type | 特征 Features | 关键词 Keywords |
|----------|---------------|-----------------|
| 木刻 Woodcut | 粗犷线条、高对比 | `woodcut`, `woodblock print`, `linocut` |
| 铜版画 Etching | 精细线条、古典感 | `etching`, `engraving`, `intaglio` |
| 丝网印刷 Screen Print | 平涂、波普风格 | `screen print`, `silkscreen`, `serigraphy` |
| 石版画 Lithograph | 柔和、绘画感 | `lithograph`, `lithography` |

**示例 Example**：
```
Japanese woodblock print style, bold outlines,
flat color areas, ukiyo-e aesthetic, traditional technique
```

---

### 粉彩 Pastel

**视觉特征 Visual Characteristics**：
- 柔和、粉质
- 丰富的色彩混合
- 纸张纹理明显
- 梦幻感

**Prompt 关键词**：
`pastel drawing`, `soft pastel`, `chalk pastel`, `powdery texture`, `blended colors`

**示例 Example**：
```
pastel portrait, soft blended colors,
powdery texture, dreamy atmosphere, impressionist style
```

## 数字艺术媒介 Digital Art Media

### 数字绘画 Digital Painting

**视觉特征 Visual Characteristics**：
- 可模拟任何传统媒介
- 完美的渐变
- 无限的撤销和修改
- 图层效果

**Prompt 关键词**：
`digital painting`, `digital art`, `digital illustration`, `Photoshop`, `Procreate style`

**示例 Example**：
```
digital painting, painterly style, rich colors,
detailed rendering, concept art quality
```

---

### 矢量艺术 Vector Art

**视觉特征 Visual Characteristics**：
- 清晰的边缘
- 平滑的曲线
- 可无限缩放
- 扁平化风格

**Prompt 关键词**：
`vector art`, `vector illustration`, `flat design`, `clean lines`, `scalable graphics`, `Adobe Illustrator style`

**示例 Example**：
```
vector illustration, clean flat design,
geometric shapes, bold colors, modern graphic style
```

---

### 像素艺术 Pixel Art

**视觉特征 Visual Characteristics**：
- 可见的像素方块
- 有限的色彩
- 复古游戏感
- 精确的像素放置

**Prompt 关键词**：
`pixel art`, `8-bit`, `16-bit`, `retro game style`, `pixelated`, `sprite art`

**示例 Example**：
```
pixel art character, 16-bit style, limited color palette,
retro video game aesthetic, crisp pixels
```

---

### 3D 渲染 3D Rendering

**视觉特征 Visual Characteristics**：
- 立体感
- 真实的光影
- 材质模拟
- 多种渲染风格

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 写实渲染 Photorealistic | 照片级真实 | `3D render`, `photorealistic`, `CGI`, `ray tracing` |
| 卡通渲染 Toon Shading | 卡通风格、硬边阴影 | `cel shading`, `toon shading`, `cartoon 3D` |
| 粘土风格 Clay | 可爱、手工感 | `clay render`, `claymation`, `3D clay style` |
| 低多边形 Low Poly | 几何简化 | `low poly 3D`, `polygonal`, `faceted` |

**示例 Example**：
```
3D render, photorealistic materials, ray-traced lighting,
octane render, high detail, studio lighting
```

## 特殊技法 Special chniques

### 厚涂 Impasto

**特点 Characteristics**：
- 厚重的颜料堆积
- 明显的笔触纹理
- 立体感强
- 梵高风格

**Prompt 关键词**：
`impasto`, `thick paint`, `textured brushstrokes`, `heavy texture`, `Van Gogh style`

**示例 Example**：
```
impasto oil painting, thick textured brushstrokes,
vibrant colors, Van Gogh inspired, expressive
```

---

### 点彩 Pointillism

**特点 Characteristics**：
- 用小点组成画面
- 光学混色
- 印象派后期技法
- 修拉风格

**Prompt 关键词**：
`pointillism`, `dotted`, `Seurat style`, `optical color mixing`

**示例 Example**：
```
pointillist landscape, tiny colored dots,
optical color mixing, Seurat inspired, impressionist
```

---

### 拼贴 Collage

**特点 Characteristics**：
- 多种材料组合
- 层叠效果
- 混合媒介
- 达达主义、波普艺术常用

**Prompt 关键词**：
`collage`, `mixed media`, `cut and paste`, `layered`, `paper collage`, `digital collage`

**示例 Example**：
```
digital collage artwork, mixed media elements,
layered textures, vintage photographs, modern composition
```

---

### 双重曝光 Double Exposure

**特点 Characteristics**：
- 两个图像叠加
- 透明融合
- 梦幻、概念性
- 摄影和数字艺术常用

**Prompt 关键词**：
`double exposure`, `multiple exposure`, `overlay`, `blended images`

**示例 Example**：
```
doube portrait, face merged with forest,
silhouette filled with nature, dreamy and artistic
```

---

### 故障艺术 Glitch Art

**特点 Characteristics**：
- 数字错误美学
- 像素位移
- 色彩分离
- 赛博朋克风格

**Prompt 关键词**：
`glitch art`, `glitch effect`, `digital distortion`, `data corruption`, `RGB split`

**示例 Example**：
```
glitch art portrait, digital distortion effects,
RGB color separation, corrupted data aesthetic, cyberpunk
```

## 摄影风格 Photography Styles

- 颗粒感
- 特定的色彩倾向
- 复古感
- 不同胶片有不同特征

| 胶片类型 Film Type | 特征 Features | 关键词 Keywords |

| Kodak Portra | 温暖肤色、柔和 | `Kodak Portra`, `warm film tones` |
| Fuji Velvia | 高饱和、鲜艳 | `Fuji Velvia`, `vivid colors`, `slide film` |
| Ilford HP5 | 黑白、高对比 | `Ilford HP5`, `black and white film`, `grainy` |
| Cinestill 800T | 电影感、霓虹晕染 | `Cinestill 800T`, `cinematic`, `halation` |


- 方形画幅

- 褪色效果
- 即时、随性感

**Prompt 关键词**：
`Polaroid`, `instant film`, `Instax`, `vintage Polaroid`, `faded colors`

**示例 Example**：
```
Polaroid photograph, white border frame,
slightly faded colors, nostalgic, casual snapshot
```

## 小结 Summary

- 媒介决定了作品的基本视觉语言和质感
- Medium determines the basic visual language and texture of artwork

- 传统媒介（油画、水彩、素描等）各有独特的视觉特征
- Traditional media have unique visual characteristics

- 数字媒介可以模拟传统媒介，也有其独特风格
- Digital media can simulate traditional media and have unique styles

- 特殊技法（厚涂、点彩、拼贴等）增加视觉趣味
- Special techniques add visual interest

- 选择合适的媒介描述能大大提升 AI 生成效果
- Choosing appropriate medium descriptions greatly improves AI generation results
