# 构图与版式 Composition and Layout

## 什么是构图 What is Composition

**构图（Composition）** 是指画面中各元素的安排和组织方式。它决定了观众的视线如何在画面中移动，以及画面给人的整体感受。

**Composition** refers to the arrangement and organization of elements in an image. It determines how the viewer's eye moves through the image and the overall feeling it conveys.

好的构图能够：
- 引导观众的注意力 Guide viewer's attention
- 创造视觉平衡或张力 Create visual balance or tension
- 强化主题表达 Reinforce thematic expression
- 提升画面美感 Enhance visual aesthetics

---

## 基础构图类型 Basic Composition Types

### 1. 中心构图 Center Composition

**特点 Characteristics**：
- 主体位于画面中央
- 对称、稳定、庄重
- 适合肖像、产品、图标

**视觉效果 Visual Effect**：
正式、直接、有力量感

**Prompt 关键词**：
`centered composition`, `symmetrical`, `central focus`, `direct view`

**示例 Example**：
```
portrait of a queen, centered composition, symmetrical,
looking directly at camera, royal atmosphere
```

---

### 2. 三分法构图 Rule of Thirds

**特点 Characteristics**：
- 画面被分成 3×3 的九宫格
- 主体放在交叉点或分割线上
- 最常用的构图法则

**视觉效果 Visual Effect**：
自然、平衡、有动感

**Prompt 关键词**：
`rule of thirds`, `off-center subject`, `balanced composition`

**示例 Example**：
```
landscape photography, rule of thirds composition,
sun positioned at upper right intersection point,
foreground interest
```

---

### 3. 黄金分割构图 Golden Ratio

**特点 Characteristics**：
- 基于 1:1.618 的数学比例
- 比三分法更精确
- 古典艺术常用

**视觉效果 Visual Effect**：
和谐、优雅、经典美感

**Prompt 关键词**：
`golden ratio`, `golden spiral`, `fibonacci composition`, `classical proportions`

**示例 Example**：
```
Renaissance painting, golden ratio composition,
classical proportions, harmonious arrangement
```

---

### 4. 对角线构图 Diagonal Composition

**特点 Characteristics**：
- 主要元素沿对角线排列
- 创造动感和方向性
- 打破静态感

**视觉效果 Visual Effect**：
动态、有能量、引导视线

**Prompt 关键词**：
`diagonal composition`, `dynamic angle`, `diagonal lines`, `movement`

**示例 Example**：
```
action scene, diagonal composition, dynamic movement,
speed lines, energetic atmosphere
```

---

### 5. 对称构图 Symmetrical Composition

**特点 Characteristics**：
- 左右或上下对称
- 强烈的秩序感
- 建筑、倒影常用

**视觉效果 Visual Effect**：
庄严、平静、有仪式感

**Prompt 关键词**：
`symmetrical`, `mirror image`, `perfect symmetry`, `bilateral symmetry`

**示例 Example**：
```
grand cathedral interior, perfect symmetry,
mirror-like reflection on floor, majestic atmosphere
```

---

### 6. 框架构图 Frame within Frame

**特点 Characteristics**：
- 用前景元素框住主体
- 增加层次感和深度
- 门、窗、拱门常用

**视觉效果 Visual Effect**：
聚焦、有深度、神秘感

**Prompt 关键词**：
`frame within frame`, `natural framing`, `archway framing`, `window view`

**示例 Example**：
```
view through ancient stone archway, frame within frame,
distant castle visible, depth and mystery
```

---

### 7. 引导线构图 Leading Lines

**特点 Characteristics**：
- 用线条引导视线到主体
- 道路、河流、栏杆等
- 创造纵深感

**视觉效果 Visual Effect**：
有方向性、深度感、引人入胜

**Prompt 关键词**：
`leading lines`, `converging lines`, `path leading to`, `perspective lines`

**示例 Example**：
```
long corridor with leading lines, perspective,
vanishing point in the distance, mysterious door at the end
```

---

### 8. 极简构图 Minimalist Composition

**特点 Characteristics**：
- 大量留白
- 元素极少
- 强调主体

**视觉效果 Visual Effect**：
简洁、现代、有禅意

**Prompt 关键词**：
`minimalist composition`, `negative space`, `simple`, `clean`, `lots of white space`

**示例 Example**：
```
single red apple on white background, minimalist composition,
lots of negative space, clean and simple
```

---

### 9. 填充构图 Fill the Frame

**特点 Characteristics**：
- 主体占满画面
- 几乎没有背景
- 强调细节和质感

**视觉效果 Visual Effect**：
冲击力强、亲密、细节丰富

**Prompt 关键词**：
`fill the frame`, `close-up`, `tight crop`, `detailed`, `macro view`

**示例 Example**：
```
extreme close-up of an eye, fill the frame,
every detail visible, hyper-detailed iris
```

---

### 10. 层次构图 Layered Composition

**特点 Characteristics**：
- 前景、中景、背景分明
- 创造空间深度
- 风景摄影常用

**视觉效果 Visual Effect**：
立度、丰富

**Prompt 关键词**：
`layered composition`, `foreground middle ground background`, `depth layers`, `atmospheric perspective`

**示例 Example**：
```
mountain landscape, layered composition,
flowers in foreground, lake in middle ground,
snow-capped peaks in background, atmospheric depth
```

## 版式与布局 Layout and Arrangement

### 视觉重量 Visual Weight

画面中不同元素有不同的"视觉重量"：

Different elements have different "visual weight":

| 重的元素 Heavy | 轻的元素 Light |
|---------------|---------------|
| 大尺寸 Large size | 小尺寸 Small size |
| 深色 Dark colors | 浅色 Light colors |
| 复杂纹理 Complex texture | 简单平滑 Simple smooth |
| 高对比 High contrast | 低对比 Low contrast |
| 实体 Solid | 透明 Transparent |

**平衡技巧 Balancing Tips**：
- 大元素配小元素
- 深色配浅色
- 复杂配简单

---

### 留白的艺术 The Art of Negative Space

**留白（Negative Space）** 不是"空"，而是设计的一部分。

**Negative Space** is not "empty"—it's part of the design.

**留白的作用 Functions of Negative Space**：
- 让主体"呼吸" Let the subject "breathe"
- 引导注意力 Guide attention
- 创造优雅感 Create elegance
- 增强可读性 Enhance readability

**Prompt 关键词**：
`negative space`, `breathing room`, `white space`, `uncluttered`, `spacious`

**示例 Example**：
```
Japanese ink painting, generous negative space,
single branch with bird, zen aesthetic, breathing room
```

---

### 视觉流动 Visual Flow

好的构图会引导观众的视线按特定路径移动：

Good composition guides the viewer's eye along a specific path:

**常见视觉流动模式 Common Visual Flow Patterns**：

**Z 形流动 Z-Pattern**：
适合有文字的设计，视线从左上到右上，再到左下到右下。
Suitable for designs with text, eye moves from top-left to top-right, then bottom-left to bottom-right.

**F 形流动 F-Pattern**：
适合信息密集的布局，视线先横向扫描，再向下。
Suitable for information-dense layouts, eye scans horizontally first, then down.

**圆形流动 Circular Flow**：
视线在画面中循环，没有明确的出口。
Eye circulates within the image, no clear exit.

mpt 关键词**：
`visual flow`, `eye movement`, `guided composition`

## 构图与风格的关系 Composition and Style Relationship

不同风格倾向于不同的构图方式：

Different styles tend toward different composition approaches:

| 风格 Style | 常用构图 Common Composition |
|-----------|---------------------------|
| 日系动漫 Anime | 动态对角线、特写 Dynamic diagonal, close-ups |
| 极简主义 Minimalism | 中心构图、大量留白 Center, lots of negative space |
| 巴洛克 Baroque | 对角线、戏剧性 Diagonal, dramatic |
| 浮世绘 Ukiyo-e | 不对称、平面层次 Asymmetrical, flat layers |
| 商业摄影 Commercial | 三分法、干净背景 Rule of thirds, clean background |
| 电影感 Cinematic | 宽幅、引导线 Wide format, leading lines |

## 小结 Summary

- 构图决定画面元素的安排方式和视觉流动
- Composition determines element arrangement and visual flow

- 不同构图类型适合不同的主题和情感表达
- Different composition types suit different subjects and emotional expressions

- 留白是设计的重要组成部分，不是"空"
- Negative space is an important part of design, not "empty"

- 风格和构图相互影响，选择时要考虑整体效果
- Style and composition influence each other; consider overall effect when choosing
