If you're a complete beginner, we recommend starting with the Tutorial Track and reading in order:

The Tutorial Track will take you from zero, systematically building your cognitive framework for image styles. Each chapter builds on the previous one, progressing step by step.

## 关于中英对照 About Bilingual Format

本手册采用中英对照的形式，原因如下：

This guide uses a bilingual format for the following reasons:

1. **Prompt 实用性**：大多数 AI 绘图工具使用英文 prompt 效果最佳，掌握英文术语能帮助你写出更有效的 prompt
2. **Prompt Practicality**: Most AI art tools work best with English prompts; mastering English terminology helps you write more effective prompts

3. **国际通用性**：艺术风格的英文名称是国际通用的，便于你查找更多资料和参考
4. **International Universality**: English names for art styles are internationally recognized, making it easier to find more resources and references

5. **学习效率**：同时学习中英文表达，一举两得
6. **Learning Efficiency**: Learning both Chinese and English expressions at once, killing two birds with one stone

每个风格条目都会提供：
- 中文名称
- 英文名称
- 常用的 prompt 关键词

Each style entry will provide:
- Chinese name
- English name
- Common prompt keywords
