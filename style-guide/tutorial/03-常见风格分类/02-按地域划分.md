# 按地域划分 Styles by Region

## 概述 Overview

不同地域的文化背景孕育出独特的视觉美学。了解这些地域风格，能帮助你在 AI 绘图中创造出具有特定文化氛围的作品。

Different regional cultural backgrounds give birth to unique visual aesthetics. Understanding these regional styles helps you create works with specific cultural atmospheres in AI art.

---

## 东亚风格 East Asian Styles

### 中国风 Chinese Style

**视觉特征 Visual Characteristics**：
- 水墨渲染
- 留白的运用
- 线条的韵律感
- 意境重于写实
- 红色、金色等传统色彩

**子风格 Sub-styles**：

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 水墨画 Ink Wash | 黑白灰、渲染、留白 | `Chinese ink painting`, `ink wash`, `sumi-e` |
| 工笔画 Gongbi | 精细、写实、勾线填色 | `Gongbi`, `meticulous`, `fine brushwork` |
| 写意画 Xieyi | 意境、简练、抽象 | `Xieyi`, `freehand`, `expressive brushwork` |
| 青花瓷 Blue and White | 蓝白配色、花纹 | `blue and white porcelain`, `Chinese ceramics` |
| 敦煌风格 Dunhuang | 壁画、佛教艺术 | `Dunhuang murals`, `Buddhist art`, `flying apsaras` |

**Prompt 示例 Example**：
```
Chinese ink wash painting, misty mountains,
pine trees, generous negative space,
traditional brushwork, zen atmosphere
```

---

### 日本风 Japanese Style

**视觉特征 Visual Characteristics**：
- 简洁、精致
- 不对称美学
- 自然元素
- 季节感
- 侘寂（Wabi-sabi）美学

**子风格 Sub-styles**：

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 浮世绘 Ukiyo-e | 木版画、平面、轮廓线 | `ukiyo-e`, `woodblock print`, `Hokusai style` |
| 日本画 Nihonga | 传统颜料、金箔 | `Nihonga`, `Japanese painting`, `gold leaf` |
| 禅画 Zen Art | 极简、书法、圆相 | `Zen art`, `enso`, `minimalist Japanese` |
| 动漫 Anime | 大眼睛、鲜艳色彩 | `anime style`, `manga`, `Japanese animation` |
| 和风 Wa-style | 传统图案、和服 | `traditional Japanese`, `kimono patterns`, `wagara` |

**Prompt 示例 Example**：
```
ukiyo-e style illustration, great wave,
bold outlines, flat color areas,
Hokusai inspired, traditional Japanese woodblock
```

### 韩国风 Korean Style

**视觉特征 Visual Characteristics**：
- 清新淡雅
- 柔和的色调
- 现代与传统融合
- 简约而精致

**子风格 Sub-styles**：

| 风格 Style | 特征 Features | 关键词 Keywords |
|-----------|---------------|-----------------|
| 韩服风格 Hanbok | 传统服饰、优雅线条 | `Hanbok`, `Korean traditional`, `elegant silhouette` |
| 民画 Minhwa | 民间艺术、吉祥图案 | `Korean folk art`, `Minhwa`, `tiger painting` |
| K-风格 K-style | 现代韩流、清新 | `Korean style`, `K-beauty`, `soft aesthetic` |
| 青瓷风格 Celadon | 青绿色、优雅 | `Korean celadon`, `jade green`, `elegant pottery` |

**Prompt 示例 Example**：
```
Korean traditional painting style, elegant crane,
soft muted colors, delicate brushwork,
peaceful atmosphere, Joseon dynasty inspired
```
