# 按用途划分 Styles by Purpose

## 概述 Overview

不同的应用场景需要不同的视觉风格。了解各种用途的风格特点，能帮助你为特定目的选择最合适的风格。

Different application scenarios require different visual styles. Understanding the style characteristics for various purposes helps you choose the most suitable style for specific goals.

---

## 插画风格 Illustration Styles

### 儿童插画 Children's Illustration

**特点 Characteristics**：
- 温暖、友好
- 简化的形状
- 明亮的色彩
- 可爱的角色

**Prompt 关键词**：
`children's book illustration`, `kids illustration`, `cute`, `friendly`, `warm colors`, `storybook style`, `whimsical`

**示例 Example**：
```
children's book illustration, cute animal characters,
warm friendly colors, simple shapes,
whimsical storybook style, heartwarming
```

---

### 社论插画 Editorial Illustration

**特点 Characteristics**：
- 概念性强
- 传达观点
- 风格多样
- 杂志/报纸使用

**Prompt 关键词**：
`editorial illustration`, `conceptual`, `magazine illustration`, `thought-provoking`, `metaphorical`

**示例 Example**：
```
editorial illustration for magazine,
conceptual metaphor, bold graphic style,
thought-provoking imagery, modern design
```

### 时尚插画 Fashion Illustration
